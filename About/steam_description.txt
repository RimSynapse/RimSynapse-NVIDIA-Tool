[h1]RimSynapse - NVIDIA Tool[/h1]
[b]Real-time VRAM monitoring and LLM performance metrics rendered directly in-game.[/b]

A crucial diagnostic tool for developers and power-users running local LLMs alongside RimWorld. Monitor GPU temperatures, active VRAM allocations, and LLM throughput directly inside the UI.

---

[h2]Key Features[/h2]
[list]
[*] [b]Live GPU Diagnostics:[/b] Real-time display of VRAM, GPU temperature, and power draw using nvidia-smi.
[*] [b]Detailed VRAM Breakdown:[/b] Tracks how much memory is allocated to model parameters, KV cache, and context window size.
[*] [b]Performance Metrics Log:[/b] Logs request latency, queue depth, and token generation speed (tokens/sec).
[*] [b]HUD Overlay:[/b] Toggleable toolbar display showing hardware health while playing.
[*] [b]Developer Tools Window:[/b] Tabbed debug window containing GPU stats, request history, and active LLM configuration.
[/list]
